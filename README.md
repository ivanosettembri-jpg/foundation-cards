# The Cabal

Gacha card game built on Foundation NFT tokens.

## Setup

```bash
npm install
npm run dev
```

## Deploy (Vercel)

Push to GitHub, then import the repo on vercel.com.
Vercel auto-detects Vite — no configuration needed.

## Card pool

Cards are loaded at runtime from:
`https://raw.githubusercontent.com/networked-art/foundation-ipfs-cids/master/token_cids.csv`

343,194 Foundation NFT tokens. Images served via IPFS.
